# Campus Priority Notifications Dashboard



![Dashboard Preview](./dashboard.png)
A responsive, high-performance React application built with TypeScript and Material-UI (MUI) to display and manage real-time campus notifications (Placements, Results, and Events).

## 🚀 Features
* **Priority-Based Visual Coding:** Cards feature distinct color-coded left borders matching the notification type (Red for Placements, Orange for Results, Blue for Events).
* **State Management:** Interacting with "Mark Read" dims down historical notifications and persists the state seamlessly using `localStorage`.
* **Dynamic Pagination & Filtering:** Fully functional Client-Side simulation of category filtering and numerical pagination tracking.
* **Webpack Module Override:** Structured using CRACO to handle strict ECMAScript module (`.mjs`) resolution rules smoothly.

## 🛠️ Tech Stack
* **Frontend Framework:** React 19 (TypeScript)
* **UI Components:** Material-UI (MUI v5)
* **Build System Override:** CRACO
* **Icons:** MUI Icons Material
